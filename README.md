# Berkay Öz — AI Product Builder

Personal product website built with Next.js 15, TypeScript, Tailwind CSS,
Framer Motion, and shadcn-style UI primitives.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

The site uses Next.js static export and writes the deployable output to `out/`.
Pushing to `main` runs the GitHub Pages workflow in `.github/workflows/deploy.yml`.
